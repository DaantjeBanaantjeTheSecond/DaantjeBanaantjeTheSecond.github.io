The mechanical brain, with it's finite creativity, can only tell you what it already imagined
This work shows the inner workings of Artificial Intelligence. Before an AI is usable it is trained on a dataset. After this training, all the possible answers it can give are already set in stone. The only thing an AI can do from that moment is look up an answer to the question you ask it.

I tried to visualize this process of looking up the already determined answer. AI's work in higher dimensional space, so to visualize the process of finding an answer in this higher dimensional space, I made an animation that collapses down into a one point. This process is done by continually folding the space until there is only one answer left.
